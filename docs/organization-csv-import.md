# 組織CSV取込フォーマット

組織管理画面の「CSV取込」から、選択中の評価期に対して組織スナップショットを一括登録できます。

## フォーマット

文字コードは UTF-8、区切り文字はカンマです。ヘッダー行は必須です。

```csv
name,parent_name
営業本部,
営業部,営業本部
営業1課,営業部
```

| 列名 | 必須 | 内容 |
| ---- | ---- | ---- |
| `name` | 必須 | 組織名。評価期内で一意にしてください。 |
| `parent_name` | 任意 | 親組織名。空欄の場合はルート組織になります。既存組織名または同じCSV内の組織名を指定できます。 |

既に同じ `name` の組織が存在する場合は、新規作成せず `parent_name` を更新します。
